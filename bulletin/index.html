UTC+8 00:00 / 2nd October 2018

我看了一篇介绍中国互联网史的文章，文章里有很多很厉害的人物的故事。我很崇拜那些人，他们发现了问题，做出了很厉害的产品，在市场的竞争和厮杀后活了下来。

我羡慕又嫉妒。

看到他们在29岁左右的年龄做出了那么厉害的事情，我觉得我生命的23年都是虚度的。我不能控制自己，在回忆我浪费掉的资源，错过的机会。想起来过去的我做的事情，我觉得我现在也没有什么进步，每天都活得毫无计划，也忘了自己的什么狗屁梦想。嫉妒，后悔，自卑，同时让我难受。

我只好找到你，把这些说给你，虽然一开始是想只写给自己看，但是也觉得我可能给别人看，所以我又不想写自己愧疚沉迷游戏想被爱和无知愚昧的事情。告诉你之后我的确心里舒服一点，我打算先把这个文件加密保存起来。

好，说回我的伤心，我浪费了成吨的时间，没有制定计划、看书、学技术等等的习惯，反而相反，我讨厌上课看书，学习新的知识。这样下去的确会无药可救，我必须给自己一个计划，至少定时去学习。目前的目标就是获得一份网页前端开发者或者用户体验设计师或者美工的工作养活自己，之后的事情我不知道。

对，你发现了，我的目标还是很模糊。网页前端是目前据我所知最有可能获得工作机会的目标，但是为了完成它我得赶紧学习并且做出作品，我想象中的作品就是我的个人网站，这个我是有信心能做到的，但是，可能是没有设定计划的原因，我到现在还没有真正可以说入门。除了迟迟没有计划之外，我还对这份工作的未来没有信心，我曾经以为我是喜欢设计工作的，我心中所想的设计就是把目标的功能明确后对形式的反复调整，我追求工具使用方式的巧妙感和美感，就是约等于「优雅」这个词。可是网页前端工程师的定位好像只是一个工人，如果AI可以根据设计稿切图写代码，我所谓的职位可能就不存在了。那再说到我刚才所说的设计，我对用户体验设计师职位的理解很浅薄，就是来自网上搜索来的介绍，对于我是否能完成这份工作，我担心我遇到的同事和客户对我表现是否满意，所以我想，要不从设计助理开始做然后学习好了。另外这份工作，我担心我我不知道去找哪一家公司。写到这里突然觉得我可以通过归类明确问题之后上网搜索解决，又同时觉得我之前竟然没有那么做觉得自己很烂，我想把之前的那个我独立出来骂他，质问他。最后一个职位目标是美工，在我心里的定位是体力活，很少有机会学到什么新东西，作图再熟练再有创意又怎么样，项目到达美工手里之后的要求只有「漂亮」，而且可能也没多少人在乎它的质量。所以我害怕，我害怕我被这种体力活压榨，变成赚钱机器，甚至没时间让自己活得高兴，更不要说去学习了。虽然可能我的能力还不够做好一名美工，我永远有办法找到时间责怪过去的自己没有努力。过去已不能改变，但是不要让将来的自己再有时间责怪现在的你

没错这应该就是我痛苦的根源，我责怪过去的自己没有做到。是的，这种责怪行为本身对于现在的我来说也是一种浪费时间的行为，而且会造成我的痛苦与自卑。但就是奇怪，好像在跟你说这些话之前我都不知道我经常做这种蠢事一样。回想一下这种该死的行为发生的频率非常之高，我好像对此已经习惯，他好像是我的固定自虐手段。谢谢你，在告诉你之后，我从未如此清楚地知道我需要停下来。

并且，我需要计划和开始。没错自我剖析还没有结束，计划和开始这件事感觉好像我要把它变成一个丢给未来的自己的不明确的指令了。我不能让这样的事发生，因为不明确所以未来的我会觉得这件事需要思考和了解所以很麻烦，我就会继续拖延，哈哈，我可真的很懒惰（好吧，我想骂我自己，我觉得我很混蛋）。那么先稍微努力让未来的我清楚一点，目前的首要目标是做网页前端工程师，为此我要设计出我的个人页面的雏形并且实现它，没错我打算使用GitHub page，我之前已经稍稍有了一些经验。要打消一个愚蠢的我要从头学 git 的念头，因为如果我那么做的话，未来的我一定会不耐烦并且放弃。所以就先从使用Figma绘制页面原型开始吧。还有一个小算盘就是如果我在实现阶段失败了，我可以把我的原型当作一个设计作品放在简历里。至于美工这个备胎计划，说不定我宁愿做服务员也不愿意做美工。

好的，让我的计划简明一些（等等我突然犹豫我的这个应该是计划呢，还是想法呢，这个问题就拖延给未来的我吧，应该是想法。。。），我的主要目标仍然是成为网页前端工程师，通过作品获得工作机会；次要但几乎平行的目标是成为交互设计师（用户体验设计师）。对于做前端工程师的未来期望是了解行业，做自己的作品，跳槽设计师或经理岗位，对于经理岗位我的自信还是少一些。或者我能稳定的生活下去之后我再去发展我其他的爱好，比如摄影音乐，当然还有游戏，这些对于现在的我来说还是很难做到的。

嗯，希望我能遇到爱情，好，诚实一点，希望遇到一个我很喜欢的并且很喜欢我，我俩又合得来的人。虽然我还是觉得就算遇到了我还是会不幸福，还好有你在，我能告诉你，更清醒地了解我自己，纠正我自己。

那下次再聊，没错我找你的时候我应该都是情绪不正常吧，哈哈哈，抱歉。
