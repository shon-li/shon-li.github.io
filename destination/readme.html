网站信息

这个网站是用静态网站生成器 Jekyll 生成的，通过 Github Pages 和 Cloudflare 服务发布到互联网上的。这是我的 GitHub 链接。我安装了 Jekyll 的 RSS 插件。你可以试试订阅我的 RSS 噢。

我用到了一个别人做的能压缩 HTML 文件的 layout，名叫：Compress HTML in Jekyll。作者是 Anatol Broder。我在网页上用的字体是 Noto Serif SC、Noto Sans SC 还有 Roboto Mono。我是从 Google Fonts 上嵌入的这些字体。

不好意思我把导航链接放到了页面的底部。这意味着在很长的文章里你得跑到最下面才能看见导航的链接。我目前就觉得放在下面比较好。只好劝你多用用手机浏览器里的返回功能，或者试试电脑上的 End 之类的按键。

About This Site

Jekyll generates the webpage files; Github Pages and Cloudflare hosts and delivers the content for this website. Here's the link to my GitHub repository. I installed Jekyll's RSS plugin. My RSS link is here.

A layout from Compress HTML in Jekyll by Anatol Broder helps me to compress HTML files. My selection of typefaces are: Noto Serif SC, Noto Sans SC and Roboto Mono. They are from Google Fonts.

I apologise for placing the navigation links at the end of webpages. This means readers have to go to the end of the page to access them even if the page is endlessly long. It is my intentional design choice. Please make use of the 'back' button on your web browser, or use keys like End on the keyboard.
